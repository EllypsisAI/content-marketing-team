# Weekly Briefing

The Monday morning experience. Show up, see what's ready, know what to do.

## Process

1. Read `memory/state.md` — what was planned for this week
2. Read all files in `queue/` — count by status (captured, drafted, in-review, approved)
3. Read `memory/content-ledger.md` — what published last week
4. Invoke the calendar skill mentally: map queue items to this week's slots (Tue/Thu/Sat + newsletter if biweekly)

### Opportunities

- If there's a timely topic from the content bank: surface it
